## 本地运行说明

1. 环境要求
   * Node.js v16+
   * Python3（用于本地服务器）

2. 安装步骤

   ```bash
   git clone [仓库地址]
   cd projects/2025-03/cascade
   npm install
   ```

3. 合约部署（可选）

   ```bash
   # 配置.env
   cp .env.example .env
   # 编辑.env添加私钥

   # 部署合约
   npx hardhat run scripts/deploy.js --network monad
   ```
